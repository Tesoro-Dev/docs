Installation

Windows

Download Tesoro for Windows from the website or GitHub. Once downloaded, extract the ZIP file to your preferred location. Open the tesoro-win64-gui folder and open the bin folder afterwards. Scroll all the way down to find the executable named tesoro-wallet-gui.exe and run it and follow the setup progress. If any problems occur make sure to run it with administrative privileges.

macOS

Download Tesoro for macOS from the website or GitHub. Once downloaded, move the .APP file to your Applications folder found in the Finder. Open the tesoro-wallet-gui app and follow the setup progress.

Linux

Download Tesoro for Linux from the website or GitHub. Once downloaded, open a Terminal window in same folder. Run the following command: chmod +x tesoro-wallet-gui. After you have granted permissions, run it with ./tesoro-wallet-gui.
